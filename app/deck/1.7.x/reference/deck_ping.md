Verify connectivity with Kong.

### Synopsis

Ping command can be used to verify if decK
can connect to Kong's Admin API or not.

```
deck ping [flags]
```

### Options

```
  -h, --help               help for ping
  -w, --workspace string   Ping configuration with a specific workspace (Kong Enterprise only).
                           Useful when RBAC permissions are scoped to a workspace.
```
