## Custom headers

The log server that receives these messages might require extra headers, such as for authorization purposes.

{% if_version gte:3.0.x %}
```yaml
...
  - name: http-log
    config:
      headers:
        Authorization: "Bearer <token>"
...
```
{% endif_version %}
{% if_version lte:2.8.x %}
```yaml
...
  - name: http-log
    config:
      headers:
        Authorization: 
          - "Bearer <token>"
...
```
{% endif_version %}
